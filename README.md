# Software-Testing-and-Quality-Assurance-A3
How to Run This Project (Local Setup)
First, open Command Prompt (CMD) in the project folder.
Run the following command:
python -m http.server 8000
After running the command, open your browser.
Go to the following URL:
http://localhost:8000/
You can also directly access the project in the browser after starting the server — there is no need to keep using CMD after the server is running.
Note
Make sure Python is installed in your system.
If port 8000 is busy, you can change it (e.g. 8001).
